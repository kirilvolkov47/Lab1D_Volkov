## Мета роботи

## Постановка задачі

Під час виконання практичної роботи необхідно розробити програму, за допомогою якої на основі методу Гоморі можна отримати цілочисловий розв'язок задачі лінійного програмування. Необхідно описати структуру програми, вхідні й вихідні дані, навести екранні форми програми, а також продемонструвати виконання як тестових прикладів, так і власного варіанта завдання.

**Зауваження.** Для виконання цієї практичної роботи використовуються варіанти завдань з практичної роботи № 1 Part B. Також знак однієї із нерівностей системи обмежень необхідно змінити на протилежний.

## Варіант 7

### Функція мети

    Z = x1 + 3x5 → max

### Обмеження

    2x1 + 2x2 + x3 + 2x4 + x5 ≤ 4
    2x1 + x2 + 3x4 + 2x5 ≤ 4
    3x1 + x3 + 2x4 + 5x5 ≤ 9

## Виконання роботи

### Використаний алгоритм

Для виконання роботи було застосовано метод Гоморі. Спочатку задача розв'язується як звичайна задача лінійного програмування без вимоги цілочисловості. Після цього перевіряється, чи всі знайдені значення змінних є цілими. Якщо так, то розв'язок уже є цілочисловим. Якщо ні, тоді будується додаткове обмеження Гоморі за рядком, у якого найбільша дробова частина у стовпці вільних членів. Це обмеження додається до симплекс-таблиці, після чого знову виконується пошук опорного та оптимального розв'язків.

Таким чином, метод Гоморі послідовно відсікає дробові розв'язки і приводить задачу до цілочислового оптимального розв'язку.

## Розробка програми

Для виконання роботи розроблено консольний застосунок мовою C#. Програма є продовженням Part A, Part B та Part C. У програмі реалізовано пошук опорного та оптимального розв'язків задачі лінійного програмування, а також метод Гоморі для отримання цілочислового розв'язку.

### Структура програми

Програма складається з п'яти класів:

- **Program** — відповідає за взаємодію з користувачем через консольне меню та зчитування вхідних даних.
- **PartA** — містить реалізацію звичайних ЖВ з першої частини роботи.
- **PartB** — містить реалізацію МЖВ з другої частини.
- **PartC** — містить реалізацію видалення 0-рядків та пошук опорного і оптимального розв'язків для змішаної системи обмежень. Код PartC скопійовано з PartB і доповнено методами для видалення 0-рядків та видалення стовпця з таблиці.
- **PartD** — містить реалізацію методу Гоморі для пошуку цілочислового розв'язку.

### Вхідні дані

- симплекс-таблиця з коефіцієнтами обмежень та Z-рядком;
- імена базисних та вільних змінних.

### Вихідні дані

- опорний розв'язок;
- оптимальний розв'язок задачі лінійного програмування;
- додаткові обмеження Гоморі;
- цілочисловий розв'язок та значення Z(max).

## Реалізація в MS Excel

Задачу було також розв'язано в MS Excel. Таблиця для Excel така сама, як у Part B. Ліві частини обмежень і значення цільової функції обчислюються за допомогою функції СУММПРОИЗВ. Відмінність Part D від Part B полягає в тому, що в надбудові «Пошук рішення» додатково задається умова цілочисловості для змінних x1, x2, x3, x4, x5.

### Результати Excel — базовий варіант

    x1 = 1
    x2 = 0
    x3 = 1
    x4 = 0
    x5 = 1
    Z(max) = 4

Результати програми збігаються з результатами Excel.
